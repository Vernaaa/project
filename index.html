📊 PM Project Dashboard

專案、子任務、臨時事項，一個畫面穩穩控住

這是一個可直接部署到 GitHub Pages 的輕量型 PM 管理工具，整合專案總覽、子任務管理、臨時事項、甘特圖、工作量分析與資料匯出。全部採用單一 HTML 檔案，部署簡單、維護輕巧，對 PM 來說比背十本佛經還實用。

專案進度儀表板

快速掌握每個專案的整體進度與目前狀態。

PM 工作量分析

查看每位 PM 目前負責的專案數量，避免資源配置失衡。

專案列表

可依關鍵字與 PM 進行篩選，快速找到目標專案。

專案明細

檢視專案資訊、子任務、臨時事項與相關操作。

專案甘特圖

以週為單位檢視主要工作排程，後續也可再擴充拖拉式調整。

專案行事曆

集中查看各專案截止日、子任務期間與臨時事項到期日，方便 PM 掌握每日重點。

跨專案臨時事項總覽

集中檢視插單、臨時需求與支援事項，避免突發工作四散在各處。

系統說明與填寫規則

協助不同 PM 以一致方式維護資料，提高專案管理品質。
基本使用建議
資料保存方式
本版本使用瀏覽器 localStorage 保存資料。換句話說，資料會保留在同一台裝置與同一個瀏覽器中。若要跨裝置共用、多人協作、權限控管或串接公司資料庫，後續需再升級為前後端版本。
目前版本：GitHub Pages 可直接部署

建議後續擴充功能

這些功能很適合下一階段持續升級，讓工具更貼近公司內部實戰需求。
